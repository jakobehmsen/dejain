Welcome to the project page of Dejain.

Dejain is a Java bytecode instrumentation library and language designed to be declarative in its expression of bytecode processing. This expression leans toward the syntax of Java.

Take a look below to se how easy it is to add an extra field and override the toString method of all classes being loaded:

class {
    +public String extraField;

    +public String toString() {
        return "extraField=" + extraField;
    }
}
